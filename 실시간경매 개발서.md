# 목적 
API를 활용해서 `전국 공영도매시장 실시간 경매정보` 를 가져와 WEB Page에서 날짜, 도매시장 선택해 경매정보를 조회하는 프로그램을 개발한다.

# 활용 API
1. `실시간 경매 API`를 활용하기위해 `./api_docs/katRealTime2-openapi.yaml` 파일을 참고한다.
2. `실시간 경매 API`를 활용하기 위해 필요한 `코드`는 `./api_docs/katCode-openapi.yaml`파일을 참고한다.

# Web page 필요사항

1. 날짜를 선택 할 수 있어야한다.
2. 도매시장을 선택 할수 있어야한다.
3. 상품 분류 코드는 아래값을 사용한다. 
  - 대분류코드 `06`
  - 중분류코드 `03`
  - 소분류코드 `36`
